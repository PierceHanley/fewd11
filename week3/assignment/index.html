Submit a project proposal and wireframes. You can use wireframe.cc and send us a link to your saved wireframes, if you don't already have a preferred image editor.

Your project can be anything you want, as long as it demonstrates the material we will have covered in the course:

  • Using meaningful HTML elements to represent different parts of the content.
  • Styling that content using logical combinations of CSS element, class, and ID selectors.
  • Making use of the box model, including floats and/or flex layouts as appropriate.
  • Referencing external resources (images, stylesheets, fonts).
  • Exhibiting dynamic and interactive behavior, implemented with JavaScript and jQuery.
